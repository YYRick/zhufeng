### 四、编程题：
- 1、有一个函数，功能是求和，但是不知道在调用的时候传递了多少个参数，有可能传递的有字符串，如果字符串能转为数字，就转后再加，如果转后是NaN，那就不加，现在请实现下这个功能（15）
```js
function test(){
    var num = 0;
    for(var i = 0 ; i < arguments.length; i++){
        if(Number(arguments[i])){
            var t1 = Number(arguments[i]);
            num += t1;
        } 
    }
    return num;
}

console.log(test(1, 2, '3', 'ccc'));
```
